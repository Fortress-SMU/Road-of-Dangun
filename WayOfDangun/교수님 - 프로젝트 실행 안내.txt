======== 게임 실행 안내 ========

빌드된 형식으로 드리고 싶었지만 아직 빌드 오류를 다 잡지 못해서 
프로젝트 형태로 업로드 했습니다. 에디터에서 실행 부탁드립니다.

독립형 게임(Standalone Game) 모드로 시작 후 
F11로 전체화면 적용하시면 플레이하기 좋습니다.


======== 프로젝트 실행 방법 ========

1. 프로젝트 압축 해제
2. RoadOfDangun.uproject 파일 우클릭 (추가 옵션 표시)
3. Generate Visual Studio project files

4. 솔루션 파일 (sln) 실행
5. DebugGame -> Development Editor 변경
6. 빌드 -> 솔루션 빌드

7. "빌드 성공" 뜨면 uproject 실행
