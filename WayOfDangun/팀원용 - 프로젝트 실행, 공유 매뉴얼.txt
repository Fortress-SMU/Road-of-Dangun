======== 프로젝트 실행 방법 ========

1. 프로젝트 압축 해제
2. uproject 파일 우클릭 (추가 옵션 표시)
3. Generate Visual Studio project files

4. 솔루션 파일 (sln) 실행
5. DebugGame -> Development Editor 변경
6. 빌드 -> 솔루션 빌드

7. "빌드 성공" 뜨면 uproject 실행
(빌드 성공 안뜨면 컴파일에 문제 있음, 반드시 해결 필요)

8. 실행 안되는 것 같아도 백그라운드에서 켜지는 중, 기다리기
(엔진 버전이 다르다는 오류 떠도 괜찮으니 확인 누르고 기다리기)

9. 에디터 설정 -> 라이브 코딩 끄기 (빌드 실패의 주요 원인)


======== 프로젝트 공유 방법 ========

해당 파일을 제외하고 모두 지운다.

- Config (설정 관련 폴더)
- Content (게임 에셋 폴더)
- Source (소스 코드 폴더)
- 프로젝트.uproject (실행 파일)

나머지 파일들은 전혀 필요없는 임시 파일들이다.

위의 4가지를 압축해서 공유한다.
